# WEB_PROJECT
Сайт для физико-математического класса.  
Содержит информацию для определения направления в крупной IT сфере.  

Используемые расширения Flask:
- Flask-Admin
- Flask-Login
- Flask-WTF
- Flask-BabelEx  
  
Управление базой данных:
- SCLAlchemy
